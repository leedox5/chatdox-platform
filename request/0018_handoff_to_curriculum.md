Title: claudox+서비스데스크 웹 뷰어 구현 관련 handoff 메모
Date: 2026.07.12
Owner: Claudox

목적
- chatdox-platform 쪽에서 SUB-C/0018 구현을 진행하면서, curriculum 저장소로 전달할 사항이 있는지 분리해 기록한다.

현재 판단
- 현재 시점에서 curriculum 저장소에 즉시 반영해야 할 새로운 수정사항은 없다.
- visibility 정책(0016)과 상품/접근 정책(0015)은 이미 확정되어 있어, platform 구현만 진행하면 된다.
- platform 쪽 SUB-C/0018 구현과 웹 확인은 완료되어, curriculum 쪽 추가 수정은 현시점에서 필요 없다.

추가 검토 트리거
- claudox/service-desk 웹 뷰어 구현 중 curriculum 문서나 가이드에 빠진 안내가 확인될 경우
- docs/curriculum 과 request 사본 운영 방식에 추가 설명이 필요하다고 판단될 경우

요청 사항
- 현재는 별도 전달 없이 platform 구현을 완료 상태로 유지한다.
- 추후 curriculum 쪽 보강이 필요해지면 Leedox가 별도 전달한다.
