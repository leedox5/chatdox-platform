## 사용자 여정 & 수익 구조

### 전환 흐름

```
1. 랜딩페이지 방문 (비로그인)
   └─ 1-2장 미리보기 (가치 제시)
      ↓
2. 회원가입 (로그인)
   └─ 7일 Trial 시작 → 1-5장 접근 (가치 체험)
      ↓
3. Trial 끝남 (7일 후)
   └─ 자동 구독 유도 (대시보드 알림 또는 이메일)
      ↓
4. 토스페이먼츠 결제 (월간)
   └─ 월 $9.99 청구 → 1-20장 전체 접근
      ↓
5. 월간 갱신
   └─ 매달 자동 갱신 (또는 구독 취소)
```

### 전환율 가정

```
기본 가정 (보수적):
- 방문자 100명
  └─ 회원가입율: 10% → 10명
      └─ Trial 활성율: 80% (자동 진행) → 8명
          └─ 구독 전환율: 50% → 4명 구독자

월 4명 신규 구독 × 평균 체류 6개월 = 월 24명 활성 구독자
```

## 수익 모델

### 월간 수익 계산 (시나리오)

**시나리오 1: 보수적 (Year 1)**
```
월 신규 가입: 100명
총 활성 구독자: 200명 (평균 2개월 체류)

월 수익 = 200명 × $9.99 = $1,998/월
연 수익 = $1,998 × 12 = $23,976/년

결제 수수료 예시 (PG 수수료 + 건당 수수료):
  결제액: $200 × $9.99 = $1,998
   수수료: $1,998 × 2.9% + ($0.30 × 200) = $57.94 + $60 = $117.94
  순수익: $1,998 - $117.94 = $1,880/월
  순수익율: 94.1%

연 순수익 = $1,880 × 12 = $22,560/년
```

**시나리오 2: 중간 (Year 2)**
```
월 신규 가입: 300명
평균 체류: 4개월 → 월 활성 구독자: 1,000명

월 수익 = 1,000명 × $9.99 = $9,990/월
연 수익 = $9,990 × 12 = $119,880/년

수수료: ~3% = $299.70/거래 수익
순 월수익 = $9,990 - ($9,990 × 0.03) = $9,690/월
순연수익 = $116,280/년
```

**시나리오 3: 목표 (Year 3+)**
```
월 신규 가입: 500명
평균 체류: 6개월 → 월 활성 구독자: 2,500명

월 수익 = 2,500명 × $9.99 = $24,975/월
연 수익 = $24,975 × 12 = $299,700/년

순연수익 (3% 수수료 후) = $290,679/년
```

## 가격 전략

### 현재 가격: $9.99/월

**선정 이유:**
1. **심리적 가격대**: $9.99는 $10보다 저렴해 보임
2. **구독 SaaS 표준**: Notion, Figma, Substack 유사 가격대
3. **낮은 진입장벽**: 월간 커피값 수준 → 구독 결정 용이
4. **높은 수익성**: 여전히 좋은 마진율 (90%+)

### 향후 가격 조정 시나리오

```
Year 1: $9.99/월 (시장 진입 & 사용자 확보)
  └─ 월 100-200명 신규 가입 목표

Year 2-3: $12.99/월 (연간 30% 인상)
  └─ 기존 사용자는 $9.99 유지 (만족도 유지)
  └─ 신규 사용자는 $12.99
  └─ 수익 +30% 효과

Year 3+: 
  - Pro 플랜 추가 ($19.99/월) → API 접근, 대량 다운로드
  - Team 플랜 ($49.99/월) → 팀 협업 기능
```

```
필요 구독자 = $333 / $9.99 = 33명

→ 월 33명 이상의 구독자가 있으면 수익!
```

## 리텐션 & 이탈 분석

### 월간 이탈율 (Churn Rate)

```
기본 가정: 5% 월간 이탈율 (업계 평균: 3-7%)

Month 1: 100명 가입
Month 2: 100 - 5 = 95명 (5% 이탈)
Month 3: 95 - 4.75 = 90명 (대략)
...
Month 12: ~55명 (11개월 후 생존율)

평균 구독 기간 = 1 / 5% = 20개월 (약 1년 8개월)
```

### 리텐션 전략

1. **이메일 마케팅**: 주 1회 새 콘텐츠 공지
2. **진도율 모니터링**: 50% 이상 진행한 사용자 재참여
3. **가치 제시**: 대시보드에서 "배운 진도" 시각화
4. **상향판매 (Upsell)**: 전문 코스나 API 플랜 제안
