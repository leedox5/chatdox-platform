## 💡 Q&A 작성 규칙

새로운 Q&A를 추가할 때:

1. **파일명**: `[순번]_[주제].md` (예: `02_subscription_model.md`)
2. **템플릿**:
   ```markdown
   # Q&A [번호]: [제목]
   
   **날짜**: YYYY-MM-DD
   **카테고리**: [카테고리명]
   
   ---
   
   ## ❓ 질문
   [질문 내용]
   
   ---
   
   ## ✅ 답변
   [상세 답변]
   
   ---
   
   ## 🎯 결론
   [요약]
   ```

3. **링크**: 새 항목을 위 목차에 추가
